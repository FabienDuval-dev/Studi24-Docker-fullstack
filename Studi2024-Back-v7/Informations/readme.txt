readme.txt


This part contains some informations to the developpers and to the user of the site.

1. For the developper

- technical_documentation.md : contains the technical documentation of the project
- ProjetStudi2024_Diagrams.pdf : the target of this pdf file is to give to the developper the UML models that have permitted to build this project.
These UML graphics can give to the developper the necessary informations to understand the modelisation of 
the initial problem : to build a website to buy some tickets for the 2024 Paris Olympic games.


2. For the user

- A user manual is provided to the users in three languages : French, English, and Spanish.
- The three files are : 
	- user_manual_sp.md
	- user_manual_en.md
	- user_manual_fr.md
